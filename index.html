The single biggest upgrade for an IRL Streamer that is directly sending their stream to twitch,kick,youtube,restream,etc is using an SRTLA relay, with an OBS instance at home actually doing the streaming. I recommend belabox cloud if you have an OBS instance at home. This will also allow connection bonding (using wifi and sim card in phone at same time), better compression for the same video quality via the HEVC codec and much quicker recovery when your stream will inevitably F. It is as simple as signing up for github, sponsoring the belabox project for $10/month, and logging in and going to SRT(LA) relays at the top. There is a media source to add in OBS and screenshots for the connection information for Moblin and/or IRL Pro. Life changing!

To remotely control OBS, RealTimeKit is excellent. Rustdesk has a full remote desktop (and you can enable permanent password). OBS Blade could be used to control OBS if you want to fiddle with WebSockets, port forwarding/VPN at home, and Moblin could also do the OBS control work if you're getting that deep. There are many additional ways to control your computer/stream, but these are tools I use.

If you want the features of belabox cloud + OBS and can't run OBS at home, there are many options for paid remote hosting for an instance, which will usually include an SRTLA relay (belabox cloud tech). TobbyTravel has made a great comparison of prices and features of most of the providers at https://comparison.dallnett.com
